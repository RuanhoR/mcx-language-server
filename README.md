# MCX Language server

## About

[MCX Language Core](//github.com/RuanhoR/mcx-core) | [Mbler Cli](//github.com/RuanhoR/mbler)

## Use

### Client

At vscode, download `mcx.language.support` plugin

### Cli

Run bash:

```bash
pnpm i -g mbler
```

### Server

```bash
pnpm i @mbler/mcx-server
```

```javascript
import * as mcx from '@mbler/mcx-server'
```
